# Amostragem-Aleatoria-em-R-Exemplos-Praticos-com-a-Base-Iris
Este projeto apresenta, de forma direta e prática, como trabalhar com diferentes tipos de amostragem em R. A ideia é mostrar como gerar amostras simples, ajustar probabilidades, controlar a reprodutibilidade dos resultados e aplicar essas amostragens na famosa base iris.
